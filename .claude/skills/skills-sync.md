# skills-sync — 구현 완료 후 문서 동기화

기능 구현이 완료된 후 SKILLS.md §0 현황표, SKILLS.md §8 로드맵 체크박스,
SKILLS_GAP_ANALYSIS.md를 실제 코드 기준으로 업데이트합니다.

## 언제 사용하나

- 기능 구현 후 PR 머지 전
- `/gap-check` 결과를 바탕으로 문서를 최신 상태로 맞출 때

## 동기화 절차

1. **현재 코드 스캔**: `/gap-check`와 동일한 방식으로 구현 상태 파악
2. **SKILLS.md §0 업데이트**:
   - 섹션별 구현률(%) 재계산
   - 상태 이모지 업데이트 (🔴 → 🟡 → ✅)
   - 마지막 업데이트 날짜 갱신
3. **SKILLS.md §8 로드맵 업데이트**:
   - 완료된 항목 `- [ ]` → `- [x]`로 변경
4. **SKILLS_GAP_ANALYSIS.md 업데이트**:
   - 상태 컬럼 업데이트
   - 전체 요약 표 구현률 재계산
   - 분석 기준일 갱신

## 주의사항

- 실제 코드를 반드시 확인한 후에만 완료 처리할 것 (가정 금지)
- 부분 구현은 구현률과 함께 "부분 (X%)" 으로 표기
- 변경 내역 요약을 사용자에게 보고 후 파일 수정 진행
